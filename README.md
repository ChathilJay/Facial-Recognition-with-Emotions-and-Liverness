# Facial-Recognition-with-Emotions-and-Liverness
Emotional Feature recognition in Human Face Software
## How to use

1. Run all cells from top to bottom.
2. Click **Start Camera**.
3. First register a person using **Register from Webcam** or **Register from Folder**.
4. Stand in front of the webcam again.
5. The system should show:
   - Name / Unknown / Blocked
   - Liveness result
   - Emotion result
   - CLF similarity score
   - TRIP similarity score
   - Final decision model and threshold

Attendance logs are saved to `phase5_logs/attendance_log.csv`.

The registered embedding database is saved to `phase5_registry/face_registry.npz`.
